QUICK START
═══════════════════════════════════════════════════════════════════════════════

Build a complete Vitte Debian package in one command:

  cd /path/to/vitte
  make -f Makefile.deb deb-complete

This automatically:
  ✓ Compiles the Vitte compiler for x86_64
  ✓ Bundles standard library
  ✓ Includes all editor support (Vim, Emacs, Nano, Geany, Tree-sitter)
  ✓ Generates documentation and guides
  ✓ Creates Debian .deb installer
  ✓ Generates checksums for verification
  ✓ Produces installation packages

Output: pkgout/vitte_2.1.1_amd64.deb + supporting files

BUILD COMMANDS
═══════════════════════════════════════════════════════════════════════════════

Show help:
  make -f Makefile.deb help

Build complete package (recommended):
  make -f Makefile.deb deb-complete

Build individual components:
  make -f Makefile.deb deb            # Build .deb installer
  make -f Makefile.deb deb-bundle     # Create distribution files
  make -f Makefile.deb verify-deb     # Verify package
  make -f Makefile.deb install-deb    # Install package

Cleanup:
  make -f Makefile.deb deb-clean

Customization:
  JOBS=8 make -f Makefile.deb deb-complete       # More parallel jobs
  OPT_LEVEL=3 make -f Makefile.deb deb           # Higher optimization
  OUT_DIR=/custom make -f Makefile.deb deb-complete # Custom output dir


VERIFICATION
═══════════════════════════════════════════════════════════════════════════════

After building, verify the package:

  cd pkgout
  bash verify-debian.sh amd64 vitte 2.1.1
  # Output: ✓ Package verification successful!

Manual verification:
  sha256sum -c pkgout/vitte_2.1.1_amd64.deb.sha256
  sha512sum -c pkgout/vitte_2.1.1_amd64.deb.sha512

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "dpkg-deb: error"
Solution: Install dpkg-dev
  sudo apt install dpkg-dev

Problem: "gcc not found"
Solution: Install build-essential
  sudo apt install build-essential

Problem: Build fails with permissions error
Solution: Ensure write access to the repository
  chmod -R u+w .

Problem: Package installation fails
Solution: Install dependencies first
  sudo apt install -f

For more help: See DEBIAN_BUILD_GUIDE.md
