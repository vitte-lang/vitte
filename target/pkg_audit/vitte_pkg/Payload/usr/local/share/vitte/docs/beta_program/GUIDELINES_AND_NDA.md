# Beta Program Guidelines and NDA

## Program rules
- Report crashes with reproduction steps within 24h.
- Keep benchmark methodology unchanged across runs.
- Submit weekly feedback using the standard questionnaire.
- Avoid publishing internal APIs before public release.

## Confidentiality scope (NDA summary)
- Covered: source snapshots, internal roadmap, benchmark dashboards, unpublished APIs.
- Allowed sharing: public docs already published on vitte-lang.org.
- Term: 24 months from signature date.

## Security handling
- Use isolated test machines/containers only.
- Do not reuse production secrets in beta environments.
- Disclose security findings privately to maintainers.

## Exit conditions
- NDA breach
- Repeated non-compliance with reporting requirements
- Unsafe testing practices
