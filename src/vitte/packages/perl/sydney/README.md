# vitte-perl-sydney

Version: `0.1.0`

HTTP request, response, router, and header helpers. This package is intentionally small, dependency-light,
and usable through `PERL5LIB` after extraction from the local Vitte registry.

## Module

`Vitte::City::Sydney`

## Public API

- `request`
- `response`
- `header`
- `route`
- `router`
- `dispatch`
- `status_text`

## Check

```sh
perl -Ilib t/basic.t
```
