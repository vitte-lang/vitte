QUICK START
═══════════════════════════════════════════════════════════════════════════════

Build a complete Vitte package in one command:

  cd /path/to/vitte
  make -f Makefile.pkg pkg-complete

This automatically:
  ✓ Compiles the Vitte compiler for x86_64
  ✓ Creates optimized universal binary
  ✓ Bundles standard library
  ✓ Includes all editor support (Vim, Emacs, Nano, Geany, Tree-sitter)
  ✓ Generates documentation and guides
  ✓ Creates macOS .pkg installer
  ✓ Generates checksums for verification
  ✓ Produces installation packages

Output: pkgout/vitte-2.1.1.pkg + supporting files

BUILD COMMANDS
═══════════════════════════════════════════════════════════════════════════════

Show help:
  make -f Makefile.pkg help

Build complete package (recommended):
  make -f Makefile.pkg pkg-complete

Build individual components:
  make -f Makefile.pkg pkg            # Build .pkg installer
  make -f Makefile.pkg pkg-bundle     # Create distribution files
  make -f Makefile.pkg verify-pkg     # Verify package
  make -f Makefile.pkg install-pkg    # Install package

Cleanup:
  make -f Makefile.pkg pkg-clean

Customization:
  JOBS=8 make -f Makefile.pkg pkg-complete         # More parallel jobs
  OPT_LEVEL=3 make -f Makefile.pkg pkg              # Higher optimization
  OUT_DIR=/custom make -f Makefile.pkg pkg-complete # Custom output dir


VERIFICATION
═══════════════════════════════════════════════════════════════════════════════

After building, verify the package:

  cd pkgout
  bash verify.sh vitte-2.1.1
  # Output: ✓ Package verification successful!

Manual verification:
  shasum -a 256 -c pkgout/vitte-2.1.1.pkg.sha256
  shasum -a 512 -c pkgout/vitte-2.1.1.pkg.sha512

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "xcode-select: error"
Solution: Install Xcode Command Line Tools
  xcode-select --install

Problem: "rsync not found"
Solution: Install via Homebrew
  brew install rsync

Problem: Build fails with permissions error
Solution: Ensure you have write access to the repository
  chmod -R u+w .

Problem: Package installation fails
Solution: Use sudo and correct target
  sudo installer -pkg vitte-2.1.1.pkg -target /

For more help: See PACKAGE_BUILD_GUIDE.md
