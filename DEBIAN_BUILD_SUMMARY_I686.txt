## 🔧 Build Process (i686-Specific)

### Compilation Stage

**32-bit Compiler Flags:**
```bash
CFLAGS="-m32 -march=i686 -O2 -fPIC"
CXXFLAGS="-m32 -march=i686 -O2 -fPIC"
LDFLAGS="-m32"
```

**What `-m32` does:**
- Tells GCC to generate 32-bit x86 code
- Links against 32-bit libc (libc6:i386)
- Produces i686 ELF binary (not x86_64)
- Requires: gcc-multilib g++-multilib

**Verification:**
```bash
file pkgout/vitte-compiled-i686
# Output: ELF 32-bit LSB executable, Intel 80386, version 1, ...
```

### Packaging Stage

**Debian Control Metadata:**
```
Package: vitte
Version: 2.1.1
Architecture: i386                 (32-bit)
Maintainer: Vitte Authors
Description: Vitte programming language (32-bit)
```

**Post-Install Script (postinst):**
- Registers alternatives for `/usr/bin/vitte`
- Sets up shell completions (bash, fish, zsh)
- Creates symlinks to stdlib

**Pre-Remove Script (prerm):**
- Removes alternatives
- Cleans up symlinks
- Preserves user configuration

**Package Contents:**
```
vitte_2.1.1_i386.deb contains:
├─ /usr/bin/vitte            (compiled i686 binary)
├─ /usr/lib/vitte/           (stdlib for 32-bit)
├─ /usr/share/vitte/         (docs, examples)
├─ /etc/vitte/               (configuration)
└─ /etc/bash_completion.d/   (completions)
```

### Distribution Stage

**Checksums Generated:**
```
vitte_2.1.1_i386.deb.sha256     (128 chars, SHA256)
vitte_2.1.1_i386.deb.sha512     (128 chars, SHA512)
```

**Documentation Generated:**

## 📋 One-Command Build

### Complete Workflow (Recommended)

```bash
make -f Makefile.deb-i686 deb-i686-complete
```

**This single command:**

```
1. CLEAN
   └─ Remove previous artifacts
      └─ rm -rf build target/debian-i686 .debstage-i686

2. COMPILE (i686)
   ├─ Check 32-bit compiler: gcc -m32 -v
   ├─ Compile with -m32 -march=i686 flags
   ├─ Generate i386 architecture binary
   └─ Verify with: file binary → "ELF 32-bit"

3. CREATE .deb
   ├─ Build Debian control structure
   ├─ Add postinst/prerm scripts
   ├─ Generate md5sums
   └─ Create: vitte_2.1.1_i386.deb

4. BUNDLE
   ├─ Generate SHA256: vitte_2.1.1_i386.deb.sha256
   ├─ Generate SHA512: vitte_2.1.1_i386.deb.sha512
   ├─ Create guides: INSTALL-DEBIAN-I686.md
   ├─ Create notes: RELEASE-DEBIAN-I686-2.1.1.md
   └─ Create script: verify-debian-i686.sh

5. COMPLETE
   ├─ Display summary of all files
   ├─ Show checksums
   ├─ Display next steps
   └─ Ready for distribution!
```

**Time:** 5-15 minutes  
**Output Location:** `pkgout/`

## 🎯 Common Commands

### Basic Workflow

```bash
# Full build
make -f Makefile.deb-i686 deb-i686-complete

# View options
make -f Makefile.deb-i686 help

# Clean
make -f Makefile.deb-i686 deb-i686-clean
```

### Customization

```bash
# Parallel jobs (faster)
JOBS=8 make -f Makefile.deb-i686 deb-i686-complete

# Maximum optimization
OPT_LEVEL=3 make -f Makefile.deb-i686 deb-i686

# Custom output directory
OUT_DIR=~/releases make -f Makefile.deb-i686 deb-i686-complete

# Combined
JOBS=8 OPT_LEVEL=3 OUT_DIR=~/releases make -f Makefile.deb-i686 deb-i686-complete
```

### Verification & Installation

```bash
# Verify checksums
cd pkgout && bash verify-debian-i686.sh i386 vitte 2.1.1

# Install package
sudo dpkg -i pkgout/vitte_2.1.1_i386.deb

# Fix dependencies
sudo apt install -f

# Verify installation
vitte --version
```

## 💻 System Requirements (32-bit)

### Minimum Hardware

- **CPU:** i686 or compatible (any 32-bit x86)
- **RAM:** 2 GB minimum, 4+ GB recommended
- **Disk:** 1 GB for build, 500 MB for final package
- **Network:** Optional (pre-cached dependencies)

### Software Requirements

**Operating System:**
- Debian 10 (Buster) 32-bit or later
- Ubuntu 18.04 LTS 32-bit or later
- Linux Mint 32-bit
- Elementary OS 32-bit
- Any Debian-based 32-bit distro

**Build Tools:**
```bash
sudo apt install build-essential gcc-multilib g++-multilib dpkg-deb make
```

**Compiler Support:**
```bash
gcc -m32 -v  # Must succeed (shows 32-bit support)
```

## 🆘 Troubleshooting Quick Ref

| Problem | Solution |
|---------|----------|
| `gcc -m32 not found` | `sudo apt install gcc-multilib` |
| `libc6:i386 not found` | `sudo apt install libc6:i386` |
| `Permission denied` | `chmod +x scripts/build-debian-i686-*.sh` |
| Build fails | `make -f Makefile.deb-i686 deb-i686-clean` then retry |
| Verification fails | Check SHA256/SHA512 in `pkgout/` |
| Too large | Use `OPT_LEVEL=3` for optimization |
