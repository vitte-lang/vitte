# VitteOS Docs

This section groups VitteOS-specific planning and status documents away from the
repository root.

## Entry Points

- `docs/vitteos/ROADMAP.md`: milestone-based VitteOS delivery plan
- `docs/vitteos/STATUS.md`: generated health/status snapshot from VitteOS checks

## Maintenance

- update roadmap changes in `docs/vitteos/ROADMAP.md`
- regenerate status with `make vitteos-status`
