# Checklist De Relecture Éditoriale

Utilisez cette liste avant de valider un chapitre.

- Cohérence vocabulaire: les mêmes concepts portent les mêmes noms.
- Ponctuation: phrases et listes terminées proprement.
- Exemples chiffrés: au moins un cas limite et un cas nominal.
- Invariants: les règles de sécurité/validité sont explicites.
- Lisibilité: chaque bloc de code a une intention claire.
- Test mental: le chapitre répond à "que se passe-t-il si l'entrée est invalide ?".
- Exercice: un `À faire` est présent avec un corrigé minimal.
