# 9. Mini-projet CLI

Prérequis: `docs/book/poche/08-tests-essentiels.md`.
Voir aussi: `docs/book/chapters/21-projet-cli.md`.

## Objectif

Assembler les bases dans un mini-projet concret.

## Exemple

CLI simple: `calc add 2 3` -> `5`.

## Pourquoi

Un projet court révèle vite les zones encore floues.

## Test mental

Quelles parties du projet relèvent des modules, des types et des erreurs?

## À faire

1. Lire les arguments.
2. Appeler une procédure métier.
3. Afficher le résultat.

## Corrigé minimal

Gardez un seul cas d'usage (`add`) avant d'ajouter des options.
