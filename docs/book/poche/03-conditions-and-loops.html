3. Conditions and Loops

Prerequisites: docs/book/poche/02-variables-and-types.html. See also: docs/book/chapters/07-control.html.

Goal

Choose a path (if) and repeat an action (loop/for).

Example

entry main at app/demo {
  let n: int = 3
  if n > 0 {
    return 1
  }
  return 0
}

Why it matters

Flow control is the central logic of a program.

Quick check

What output do you get if n = 0?

Try this

  1. Edit condition.
  2. Test 3 input values.
  3. Note the exits.

If you're stuck

Always write the nominal case first, then the limiting case.
