5. Data Structures

Prerequisites: docs/book/poche/04-procedures.html. See also: docs/book/chapters/08-structures.html.

Goal

Group related data into a structure.

Example

form User {
  id: int
  name: string
}

Why it matters

Structures avoid scattered and confusing settings.

Quick check

What happens if you add email: string?

Try this

  1. Create User.
  2. Add a field.
  3. Adapt the user code.

If you're stuck

Start with 2 or 3 fields maximum, then evolve little by little.
