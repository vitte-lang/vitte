# Vitte Tout-En-Poche

Ce parcours est la version simple du livre.

Objectif: apprendre vite, sans surcharge, avec des exemples courts.

Ce format couvre maintenant tous les chapitres du livre principal (`docs/book/chapters/*.md`) en version simplifiée.

Comment lire:
1. Suivre les chapitres dans l'ordre.
2. Faire tous les mini-exercices.
3. Revenir ensuite au livre complet (`docs/book/chapters/`) si besoin.

Point d'entrée: `docs/book/poche/SOMMAIRE.md`.
