0. How to Use This Mini-Book

Level: beginner · Reading time: 3 min · Prerequisite: book/chapters/checklist-avant-publication.html · Last review: 2026-05-09

TL;DR (5 lines)

  • Read the core idea first.
  • Understand one concept at a time.
  • Run small examples.
  • Fix errors early.
  • Move to next chapter only when clear.

Frequent mistakes

  • Skipping prerequisites.
  • Reading without trying examples.
  • Fixing too many errors at once.

Prerequisites: none. See also: book/chapters/00-foreword.html.

Goal

Read Vitte in short sequence, without unnecessary jargon.

Example

Simple rule: 1 chapter = 20 minutes maximum.

Why it matters

A short, regular rhythm improves memorization.

Quick check

Can you explain in 2 sentences what you learned without rereading the chapter?

Try this

  1. Block out 20 minutes a day.
  2. Read a chapter.
  3. Do the exercise.

If you're stuck

If you only have 10 minutes, read only Goal + Example + To Do.
