0. How to Use This Mini-Book

Prerequisites: none. See also: docs/book/chapters/00-foreword.html.

Goal

Read Vitte in short sequence, without unnecessary jargon.

Example

Simple rule: 1 chapter = 20 minutes maximum.

Why it matters

A short, regular rhythm improves memorization.

Quick check

Can you explain in 2 sentences what you learned without rereading the chapter?

Try this

  1. Block out 20 minutes a day.
  2. Read a chapter.
  3. Do the exercise.

If you're stuck

If you only have 10 minutes, read only Goal + Example + To Do.
