# Glossaire Vitte

Ce glossaire centralise les notions de base pour éviter de les redéfinir dans chaque chapitre.

- `int`: type des entiers signés, utilisé pour les calculs sans décimales.
- `string`: type du texte.
- `bool`: type logique (`true` ou `false`).
- `proc`: déclaration d'une procédure avec entrées et type de sortie.
- `entry`: point d'entrée exécutable du programme.
- `let`: création d'une variable locale.
- `set`: mise à jour d'une variable existante.
- `if`: garde conditionnelle.
- `match`: sélection de branche selon une forme de valeur.
- `case`: branche d'un `match`.
- `otherwise`: branche de secours d'un `match`.
- `give`: retour immédiat depuis une branche.
- `return`: fin d'exécution avec un code de sortie.
- `form`: structure de données nommée.
- `pick`: type à variantes fermées.

Conseil de lecture: si un mot-clé vous bloque pendant un chapitre, revenez ici, puis reprenez l'exemple immédiatement.
