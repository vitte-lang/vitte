Vitte Keywords

This directory is Vitte's lexical reference. Each keyword now has an encyclopedic page designed like a compact wiki entry: definition, semantics, canonical syntax, valid example, counter-example, diagnostics, and contextual links.

The sheet format has been simplified for reference reading: no quiz, no pedagogical CTA, only detailed and navigable documentation.

Reading frame for each sheet:

  • Overview with a short summary of the keyword's role.
  • Definition and canonical syntax.
  • Detailed semantics to understand what the keyword truly changes.
  • Vitte example and comparison with C when it clarifies intent.
  • Recommended uses, limits, and common pitfalls.
  • Invalid example and its associated diagnostic family.
  • Common compilation errors.
  • Neighbor keywords, linked chapters, and complementary references.

Useful navigation:

  • Coverage view: docs/book/chapters/keywords/couverture.html.
  • Recommended order: docs/book/chapters/keywords/parcours.html.
  • Commented gallery: docs/book/chapters/keywords/index.html.
  • Thematic packs: docs/book/chapters/keywords/packs-apprentissage.html.
  • Rare or currently unused surfaces: docs/book/chapters/keywords/non-utilises.html.
  • Shared diagnostics: docs/book/chapters/keywords/erreurs-compilateur.html.
  • Full chapter set: docs/book/chapters/.
