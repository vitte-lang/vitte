Keyword all

Level: Beginner.

Prerequisites: docs/book/chapters/keywords/share.html, docs/book/chapters/09-modules.html. See also: docs/book/chapters/27-grammar.html, docs/book/chapters/31-build-errors.html.

Quick reading

Concrete reference: all is used to decide an execution path in a readable and verifiable manner. Use all when it is used to make program behavior readable; avoid it if it makes no observable difference on the branch or output.
