# Bytecode Format

## Opcodes

- `LOAD_CONST` - Charger une constante
- `LOAD_VAR` - Charger une variable
- `STORE_VAR` - Stocker une variable
- `ADD`, `SUB`, `MUL`, `DIV` - Opérations arithmétiques
- `JMP` - Saut inconditionnel
- `JMP_IF_FALSE` - Saut conditionnel
- `CALL` - Appel de fonction
- `RETURN` - Retour
- `PRINT` - Affichage
