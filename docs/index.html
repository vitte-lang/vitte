About Vitte

Vitte is a modern systems programming language focused on performance, safety, and clarity. It provides a clean syntax, strong typing, and a powerful standard library designed for real-world applications.

Quick Example

space hello/world proc main() -> int { let msg: string = "Hello, Vitte" give 0 }
