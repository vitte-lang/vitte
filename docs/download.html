Before you install

Confirm which path you want: a release package, a local build, or a quick check of the current compiler behavior. The page keeps those choices separate so you can move straight to the action that matters.

  1. Read the current notes if you want the context.
  2. Install or build the toolchain if you want to run it locally.
  3. Use the diagnostics page if you need to verify a result or decode an error.
