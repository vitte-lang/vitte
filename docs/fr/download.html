Compilateur Vitte

La chaîne d'outils visible suit désormais de bout en bout la surface du compilateur Vitte : le pilote réside dans la source Vitte, la grammaire et l'analyseur restent alignés, et les vérifications du dépôt s'attendent à une arborescence uniquement Vitte.

Avant d'installer

Confirmez le chemin souhaité : un package de version, une version locale ou une vérification rapide du comportement actuel du compilateur. La page sépare ces choix afin que vous puissiez passer directement à l'action qui compte.

  1. Lisez les notes actuelles si vous voulez le contexte.
  2. Installez ou créez la chaîne d'outils si vous souhaitez l'exécuter localement.
  3. Utilisez la page de diagnostics si vous devez vérifier un résultat ou décoder une erreur.

Notes de version – Version 2.1.1

Date : 18 février 2026

Référence de publication : Vitte2026_02

  • Installateur : vitte-2.1.1.pkg
  • Désinstallateur : vitte-uninstall-2.1.1.pkg
  • Compris : binaires vitte, vittec, pages de manuel, complétions de shell, ressources d'exécution et d'éditeur.
  • Correctif : l'essai internal_module_denied revient au diagnostic attendu E1016.

Chemin recommandé

  • Installer : déposez les outils et vérifiez l'environnement.
  • Compiler : compiler, exécuter les tests, préparer la sortie.
  • Vérifier : validez un exemple et lisez le diagnostic.

Installer

  • brew install vitte
  • make build
  • make install

Compiler

  • make build
  • make test
  • make core-language-gate

Vérifier

  • vitte check examples/first_project.vit
  • vitte explain E0001
  • make book-links-check

État actuel

  • Le travail actuel du compilateur suivi concerne uniquement Vitte, aucune source back-end héritée ne restant dans la surface active.
  • CI est vert sur les chemins rapides et stricts, de sorte que la page de téléchargement reflète une chaîne d'outils fonctionnelle plutôt qu'un état d'espace réservé.
  • Les prochaines vérifications évidentes sont les documents, les notes du pilote et le journal des actualités, qui pointent tous vers la même surface du compilateur.
