Maturité du forfait

Cette page définit les labels de maturité utilisés par docs/PACKAGE_INDEX.md.

Étiquettes

  • documented-experimental: surface publique visible avec documentation, mais devrait toujours changer jusqu'à ce qu'elle soit explicitement promue.
  • internal-by-default: surface de package local du référentiel qui ne doit pas être traitée comme une promesse publique de compatibilité.

Règle

Les packages sont internes par défaut. Un package devient documenté expérimental uniquement lorsqu'il possède une entrée de documentation ou d'index API intentionnelle et qu'il est répertorié par le générateur d'index de package.
