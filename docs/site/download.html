Chemin recommandé

Installer poser les outils et vérifier l'environnement
→
Builder compiler, lancer les tests, préparer la sortie
→
Vérifier valider un exemple et lire les diagnostics

Installer

  • brew install vitte
  • make build
  • make install

Builder

  • make build
  • make test
  • make core-language-gate

Vérifier

  • vitte check examples/first_project.vit
  • vitte explain E0001
  • make book-links-check

Ce qu'il faut regarder

  • Installation : l'outil se pose sans étape cachée.
  • Build : la chaîne complète doit rester lisible et reproductible.
  • Vérification : un exemple simple suffit pour confirmer que tout répond.

Conseil

Installer d’abord, valider ensuite, puis lire le book si le comportement n’est pas clair.
