# Migration Markdown

Les pages HTML du répertoire `docs/` ont été supprimées et remplacées par des pages Markdown.

## Ce qui change

- Les pages du site vivent maintenant en `.md`.
- Les liens internes pointent vers les pages Markdown correspondantes.
- La navigation vise désormais `diagnostics.md`.

## Ce qui ne change pas

- La structure des sections reste la même.
- Les chemins de lecture du livre restent inchangés.
- Les diagnostics et la documentation technique restent accessibles au même contenu.
