Markdown migration

The main pages of the site remain in Markdown, but the entry and index pages now use a raw HTML layout.

What changes

  • Public home pages use simple HTML tables and sections.
  • Internal links continue to point to the corresponding Markdown pages.
  • Navigation remains organized around docs/book.

What doesn't change

  • The structure of the sections remains the same.
  • The book reading paths remain unchanged.
  • Diagnostics and technical documentation remain accessible to the same content.
