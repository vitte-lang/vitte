Vitte Governance Pack

This folder defines the long-term governance and maturity baseline for Vitte.

Documents

  • long_term_ecosystem_maintenance.md: deprecation workflow, compatibility and migration policy, release cadence, windows support, issue triage process.
  • platform_governance_complete.md: architecture review process, RFC process, diagnostics policy, stdlib/backend acceptance policy, stability guarantees.
  • platform_maturity_baseline.md: platform maturity baseline and objective evidence required to claim platform readiness.

Rule

A process is not considered active until:

  1. it is documented in this folder,
  2. it has at least one verifiable gate in CI/scripts,
  3. it has an owner role and review cadence.
