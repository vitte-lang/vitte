# Vitte Stdlib API

This file is generated from stdlib sources by `tools/stdlib/generate_api_docs.py`.
Run the generator when public modules are added or renamed.

## Current Stabilized Surface

- `core.slice`: slices, checked indexing, chunks, windows, binary search, stable and unstable sort.
- `core.iterator`: iterator contracts and Vitte implementations for common terminal operations.
- `alloc.vec`: heap vector growth, capacity, push, get, drop, and iteration contracts.
- `alloc.string`: owned UTF-8 strings, reserve, push, concat, and checked slicing.
- `std`: OS-backed I/O, filesystem, environment, time, and process modules.
