Bootstrap Stages

  • Stage0 role: trusted seed entrypoint and bootstrap contract.
  • Stage1 role: first self-produced compiler artifact.
  • Stage2 role: recompiled compiler for consistency checks.

System dependencies:

  • POSIX shell
  • cc toolchain
  • linker (ld)
  • optional clang for strict profile

Bootstrap example

make stage0
make stage1
make stage2
  • make stage0 builds the initial seed compiler (vittec0.seed).
  • make stage1 produces the first self-hosted compiler and verifies that it can rebuild the project.
  • make stage2 reconstructed a second time to confirm the consistency of the artifacts.
