# Runtime - Vitte Standard Library

The standard library providing essential functionality for Vitte programs.

## Structure

### std/
Complete Vitte standard library
- **core/**: Core library (no allocator required)
- **alloc/**: Heap allocation and collections
- **std/**: Full standard library with OS support
- **runtime/**: Runtime support (unwinding, panic)
- **extra/**: Optional components

## Purpose

The runtime provides the essential APIs and functionality that Vitte programs need to run effectively.
