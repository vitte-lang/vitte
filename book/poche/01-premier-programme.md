# 1. Premier programme

Prérequis: `book/poche/00-comment-lire.md`.
Voir aussi: `book/chapters/01-demarrer.md`.

## Objectif

Lancer un programme qui renvoie une valeur.

## Exemple

```vit
entry main at app/demo {
  return 0
}
```

## Pourquoi

Ce pattern est la base de tous les exercices suivants.

## Test mental

Que se passe-t-il si vous remplacez `0` par `1`?

## À faire

1. Copier le snippet.
2. Compiler.
3. Vérifier le retour.

## Corrigé minimal

Le code ci-dessus suffit: il est valide, lisible et exécutable.
