# Type System Advanced 1-20

This directory anchors the tracked fixtures for `make type-system-advanced-20`.

The gate covers the first advanced type-system slice:

- local and bidirectional inference
- unification and internal substitutions
- generic functions, types, methods, defaults, bounds, and cycles
- MIR/IR generic instantiation evidence
- traits and explicit impl checks

Generated proof artifacts are written under `target/type_system/advanced_20/` and `target/reports/type_system_advanced_20.*`.
