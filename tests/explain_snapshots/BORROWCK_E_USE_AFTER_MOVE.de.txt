error code: BORROWCK_E_USE_AFTER_MOVE
lang: de
phase: borrowck
message: Wert verwendet after Verschiebung
summary: Wert verwendet after Verschiebung.
cause: A value is used after ownership has moved away from its binding.
step 1: Show where the value is created, where it is moved, and where it is reused after the move.
fix: use the value before the move, borrow it instead of moving it, or create a new value
example: let value: string = "vitte"
