COMPILER CLI
================================================================================

./build/debug/vittec [OPTIONS] <input.vitte> [-o output]

OPTIONS:
  --tokens         Emit token stream
  --emit-c         Emit C code
  --json-diag      JSON diagnostics
  -o <file>        Output file
  --help           Show help
  --version        Show version

EXAMPLES:
  ./build/debug/vittec --tokens examples/hello.vitte
  ./build/debug/vittec --emit-c examples/hello.vitte -o /tmp/hello.c
