# runtime/bin

Executables for the Vitte runtime layer (C17).

Typical binaries:
- vitte-run      : run bytecode / native artifacts
- vitte-vm       : VM host (bytecode interpreter)
- vitte-disasm   : disassembler for bytecode / object format
- vitte-pack     : pack/unpack runtime images
- vitte-test     : runtime test harness (golden)
- vitte-bench    : micro benchmark runner
