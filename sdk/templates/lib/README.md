# Library Template

## Overview

This template provides a structure for creating a reusable library.

## Files

- `muffin.muf` - Package manifest
- `src/lib.vit` - Main library file
- `src/` - Library modules
- `tests/` - Test files
- `examples/` - Usage examples

## Building

```bash
vitte-build --lib
```

## Publishing

```bash
vitte publish
```

## Documentation

```bash
vitte doc --open
```
