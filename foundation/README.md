# Foundation - Core Vitte Runtime

Core runtime foundations and development utilities.

## Structure

### core/
Core language components, error handling, and diagnostics
- Error infrastructure
- Logging and debugging support
- Compiler driver and session management
- Public item tracking

### utils/
Essential data structures, memory management, and utilities
- Data structures (hashmaps, vectors, etc.)
- Arena allocators
- Utilities for common operations
- File system operations

## Purpose

The foundation layer provides the minimal infrastructure needed for the compiler to operate and for programs to run.
