# Support

This document explains how to get help with Vitte.

## Community Support

- Open a GitHub issue with a minimal reproduction
- Include OS, toolchain version, and steps to reproduce

## Bug Reports

Please include:
- A minimal example (`.vit` file if possible)
- Expected vs actual behavior
- Compiler version / commit hash
- Platform (OS, CPU)

## Feature Requests

- Explain the use case
- Provide an example of the desired syntax or behavior

---

## FAQ

**Where do I ask questions?**  
Open an issue with a clear title.

**Do you provide commercial support?**  
Not at this time.
